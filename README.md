# next.js-boilerplate

## Setup commands

```shell
pnpm create next-app -- --ts . # create next.js project
pnpm i -D prettier # setup formatter

pnpm i -D husky
pnpm dlx husky-init && pnpm install

pnpm i -D jest ts-jest @types/jest
pnpm i -D @testing-library/react @testing-library/jest-dom jest-environment-jsdom @types/testing-library__jest-dom @testing-library/user-event @types/testing-library__user-event
```
